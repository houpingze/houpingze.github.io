递归基本思想

2020-09-05 课堂笔记

递归的两大要素：

递推关系：

根据某种方法，对问题进行拆分【递】

终止条件：

到达某种条件，问题可以直接求解，此时到达递归的边界，然后依次组合出原来的大问题的解。【归】

代码讲解：

2404：

#include<iostream>
using namespace std;
int f(int n){
    if(n==1) return 1;
    return f(n-1)+n;
}
int main(){
    int n;
    cin >> n;
    cout<<f(n);
}

f(n)f(n)f(n)表示前n个数的和
递推关系：f(n)=f(n−1)+nf(n)=f(n-1)+nf(n)=f(n−1)+n
边界:f(1)=1f(1)=1f(1)=1
